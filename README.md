# PowerPINN-DB-Cauchy ⚡️

本项目旨在构建一个面向复杂电力系统的高维增强型混合物理信息神经网络（PINN）求解框架。
核心创新：**深浅层混合 Cauchy 网络架构** + **DB-PINN 双重平衡损失优化策略**。

## 👥 团队成员与分工
* **杨泽坤 (总体负责人)**: 实验统筹、PowerPINN 基础框架复现、论文统稿。
* **孙修驰 (网络架构师)**: 核心攻坚 compleX-PINN / Cauchy 激活函数模块、文献梳理。
* **汤震雨 (损失优化与对标)**: 核心攻坚 DB-PINN 双重平衡模块、PI-KAN 引入及对比实验。

## 📂 目录结构说明
* `papers/`: 核心参考论文及组会阅读笔记。
* `src/`: 核心源代码（模型、数据生成、训练循环）。
* `configs/`: 实验超参数配置文件 (`.yaml` 或 `.json`)。
* `data/`: 训练与测试用的轨迹数据集（已在 `.gitignore` 中忽略，不可直接 push）。
* `experiments/`: 模型权重及中间训练日志。
* `figures/`: 结果可视化图表。
* `docs/`: 组会记录、答辩汇报与进度表。

## 📅 协作规范
1.  **组会机制**: 每周日晚统一复盘进度，对齐代码接口。
2.  **分支管理**: 各自负责模块在独立分支（如 `feature-cauchy`, `feature-db`）开发，确认无误后由队长 Merge 入主分支。
3.  **环境统一**: 统一使用 PyTorch 2.x，详见后续提交的 `requirements.txt`。
